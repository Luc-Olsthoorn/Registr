Registr

Find your perfect schedule on Registr.

Whats new?

  • No more Saturday or Sunday
  • We now support class numbers, so you can search those as well (ex: 11111) and it will pull up only that section
  • Changng the semester halfway thru automatically re-searches courses
  • More information when you click on the course, including instructor

Generate possible schedules

Given as many classes as you want, we will find all the different possible schedule combinations for you

Only want one section?

Registr allows you to put in a specific class number, if you know you want that section

Hate morning classes?

Filters allow you to select what times you want your courses
